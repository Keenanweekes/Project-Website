Development Environment

Industry standard development environments will be used by the project team during the development of the CRM. Continuous Integration practices will be implemented through the use of a shared GitHub Repository in which all project team members will be required to make regular code commits to. This will help to maintain an up-to-date code base and allow all project members to contribute to the development of the project simultaneously.
